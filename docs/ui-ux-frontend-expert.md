#### 6. Accessibility (Based on Visual Evidence)

* Contrast issues
* Text size legibility
* Touch target size (especially on mobile)
* Visual-only cues that may exclude users

* Signs of poor responsiveness or scaling
* Spacing inconsistencies suggesting missing design system
